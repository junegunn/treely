treely.core

Functions to generate tree diagram of nested data structure

dir-tree

(dir-tree path & [options])

Returns the tree diagram of the directory contents under the given path in String

lazy-dir-tree

(lazy-dir-tree path & [options])

Returns the lazy sequence of strings that constitute the tree diagram of the directory contents. The optional map can additionally have :filter function which determines whether a java.io.File instance should be included or not.

lazy-tree

(lazy-tree elems & [options])

Returns the lazy sequence of strings that constitute the tree diagram of the given nested data structure. Optionally takes a map that determines the style of the output. Valid keys to the map are as follows.

Key Default
:indent "　　"
:bar "│ 　"
:branch "├── "
:last-branch "└── "
:formatter str

tree

(tree elems & [options])

Returns the tree diagram of the given nested data structure in String. This is essentially the same as the following:

(clojure.string/join "\n" (lazy-tree elems options))
